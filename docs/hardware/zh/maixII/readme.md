# MaixII

继经典的K210 MAIX-I系列 AIOT 板卡之后，Sipeed 继续推出多款可运行完整 Linux 系统的边缘计算板卡，将以MAIX-II，MAIX-III 依次命名。

MAIX-II 系列板卡目前有 MaixII Dock、MaixSense 和 MaixII-S 三款。其中以全志 V833 为主控的 MaixII-S 主要用于商业定制用途。
