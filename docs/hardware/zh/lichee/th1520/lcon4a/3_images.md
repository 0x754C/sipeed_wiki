## Sipeed官方镜像

Sipeed 官方镜像基于 Debian 系统修改适配。 

默认镜像的帐号密码配置如下：
账户： `sipeed`，密码：`licheepi`；
账户：`debian`，密码： `debian`；
root 账户默认没有设置密码。
